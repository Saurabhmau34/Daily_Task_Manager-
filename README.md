# 📋 Smart Task Manager

A modern **Task Manager** built using **HTML, CSS & JavaScript**. This project helps you manage daily tasks while also demonstrating important JavaScript DOM concepts through interactive examples.

---

## ✨ Features

- ✅ Add New Tasks
- ✏️ Edit Tasks
- 🗑️ Delete Tasks
- ✔️ Mark Tasks as Completed
- 📊 Live Task Statistics
- 🌙 Dark / Light Mode
- 📱 Responsive Design
- 🧠 Attributes vs Properties Demo
- 🌐 Browser Rendering Pipeline
- 🖱️ Event Propagation Demo
