Attributes vs Properties

What is the Difference?

Attribute is the value written inside the HTML tag. It can be accessed using getAttribute() and changed using setAttribute().

Property is the current value of an element inside the JavaScript DOM object. It changes whenever the user interacts with the element or JavaScript updates it.

Attribute Property
Stored in HTML Stored in DOM Object

💡 Try it below: Type something in the input box, then click Show Attribute and Show Property. You'll notice that the Attribute usually remains the original HTML value, while the Property always reflects the current value of the input.
